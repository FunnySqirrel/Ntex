Установка необходимых для работы сервера библиотек:

LibJpeg - Библиотека была выбрана, так как является наиболее легковесной и достаточной для отзеркаливания изображения.

1) Откройте терминал в папке jpeg-9e.

2) Введите команду "./configure"
	2.1) Если возникнет ошибка "bash^M: bad interpreter: No such file or directory" - введите команду "sed -i -e 's/\r$//' *". Это заменит символ переноса строки с CL RF, используемый в Windows на символ переноса каретки используемый в Linux.

3) Введите команду "sudo make install"

Сборка программы:

1) Откройте терминрал в корневой папке.

2) Введите команду "cmake ."

3) Введите команду "cmake --build ."

Полученный исполняемый файл на данный момент отзеркаливает по горизонтали файл in.jpg. Результат выводится в файле out.jpg.
